## 글 추가 등 블로그 수정 후에 반영하는 방법

```bash
bundle update
bundle install
```



## 로컬에서 블로그 띄우는 방법

```
bundle install
bundle exec jekyll serve
```



## Ruby 버전 변경 당시에 사용헀던 방법

```bash
rbenv install -l
rbenv install 2.6.6
rbenv local 2.6.5
gem install bundler
bundle install
```
